# マルチプロジェクト調査

## 目的

* 自作ライブラリを自作アプリケーションから使用する
* 複数プロジェクトで共通設定ファイルを利用することを考え、プロジェクト外に設定ファイルを配置する

## ディレクトリ構成

```
root
  ├ batch    : Javaアプリケーション実行用バッチ
  ├ conf     : 設定ファイル置き場
  └ projects
      ├ common   : Java ライブラリ
      ├ pj1      : common, pj2 を使用している Java アプリケーション
      └ pj2      : common を使用している Java ライブラリ
```

# 課題
pj1 の jar ファイル作成時に、common と pj2 の jar を作成できない  
（pj1 の jar タスク完了後に依存プロジェクトの jar が作成されるため、libs ディレクトリに依存プロジェクトの jar を含められない）  

とりあえず、jar を集めるバッチ（projects/pj1/gatherJars.bat）を別に作成した
